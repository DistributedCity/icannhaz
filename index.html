Frequently Axed Questions:

You got questions, I got answers!

  • What platforms does this run on? - This first alpha release runs on Linux and Windows. OS X is being tested and works fine, just not packaged for released yet.
  • Can I use this in Opera or Chrome or IE? - We made the system plug and play for Firefox. Once it is running in Firefox you can use other browsers if you set them up in a special way. Beyond the scope of this FAQ.
  • Isn't this the same thing as MafiaaFire Browser Plugin? - Haha. Not quite. MaffiaaFire was a nice try, and credit to them for their effort. That system is still centralized turning MafiaaFire plugin developers into a backup centralized DNS system. Namecoin and the Icanhazz.bit system is fully distributed, encrypted, digitally signed p2p infrastructure.
  • What does the plugin do technically? - Well not to get technical but it runs a namecoin client, a node.js DNS server, a node.js SOCKS proxy server, and redirects .bit domain requests through the local proxy doing internal .bit lookups and letting you use the browser as normal.
